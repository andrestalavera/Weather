# Instructions
> This project is an exercice.

Create a .Net web responsive application. that, after interviewing a web service of your choice, displays the results in a **professional** grid.
- Use a web service (existing or not)
- Displays results in a **professional** grid
- There no need to create a database

# Choices
Weather forecast API and PWA.

| Technology       | Choice                    |
|------------------|---------------------------|
| Web API          | ASP.NET Core 5            |
| Mobile App       | Blazor.NET PWA            |
| Database         | InMemory (with Ef Core 5) |
| Langage          | C# 8.0                    |
| Grid             |                           |

# Build, test and run
## Requirements
- .NET 5.0.200 SDK 

## Tools
- `dotnet-cli` 
- Visual Studio

## Build
Use `dotnet build` at root.

## Tests
> No tests are implemented

## Run
Launch `Weather.Api` and `Weather.UI` with `dotnet run` command.
